# Sanitized Run Examples

This folder is reserved for public-safe examples generated by Agentic Commerce Swarm.

Do not place raw business runs here.

## Safe example data

A safe example can include:

- fictional company name;
- fictional product;
- fictional website sections;
- generated strategy;
- generated copy;
- QA report;
- rejected output example;
- final human approval notes.

A safe example must not include:

- real customer names;
- real company data;
- phone numbers;
- private emails;
- private local paths;
- credentials;
- raw transcripts;
- sensitive business strategy.
